PROJET SIERPINSKI

Le projet sur le triangle de Sierpinski consiste à créer un programme en Python qui réalise une fractale triangulaire en fonction du nombre d'itérations (nombre de répétitions) donné.

Pour réaliser ce projet j'ai écrit un programme en Python en utilisant le module Turtle pour créer et afficher la fractale. Ce programme est constitué de quattre fonctions contenant des boucles bornées. Je n'ai pas eu beaucoup de problèmes pour réaliser ce programme, juste un petite complication dans la fonction 'regleSierpinski'.
